General Comments

This was not an easy exam. I don't think it was too long -- simply broad and it made you think. Scores ran the gamut, but there were a few in the 90's, which is very impressive. Here is my breakdown of assigning letters to the numbers.
  • 80 -100: A+
  • 65 - 80: A
  • 60 - 65: B+
  • 50 - 60: B
  • 45 - 50: C+
  • 35 - 45: C
  • 20 - 35: D
  • 0 - 20: F
